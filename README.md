# Senior Living DR — Coming Soon

A simple landing page for Senior Living DR's investment platform.

## Deploy to Vercel

1. Push this project to a Git repository (GitHub, GitLab, or Bitbucket).
2. Go to [vercel.com](https://vercel.com) and sign in.
3. Click **Add New Project** and import your repository.
4. Vercel will auto-detect the static site — click **Deploy**.

Or deploy from the CLI:

```bash
npm i -g vercel
vercel
```
